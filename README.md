# Retro Sprint

Adds sprinting to b1.7.3 via [Ornithe](https://ornithemc.net/) + [RetroAPI](https://github.com/matthewperiut/retroapi) using Mixins, with a [Babric](https://babric.github.io/) build also published for Babric users!
Support for [STAPI](https://modrinth.com/mod/stationapi) Keybinds
Support for Unitweaks FOV Slider
Great compatibility with other Ornithe and Babric mods!
Works without [STAPI](https://modrinth.com/mod/stationapi)
